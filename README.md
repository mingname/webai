# UI原型 - 使用说明

## 功能特性

### 登录页面
- 用户名和密码输入
- 记住密码功能
- 密码显示/隐藏切换
- 表单验证

### 首页
- 双Tab切换（支持左右方向键切换）
- 响应式卡片布局
- 移动端侧边导航栏

### 设备预览
- Web端和移动端切换预览
- 实时响应式布局调整

## 切图功能

### 使用方法

1. **选择页面**：登录到想要的页面（登录页或首页）

2. **选择设备模式**：
   - 点击桌面图标（🖥️）切换到Web端模式
   - 点击手机图标（📱）切换到移动端模式

3. **点击截图按钮**：点击右上角的绿色相机图标按钮

4. **自动下载**：截图会自动下载为PNG文件

### 截图文件命名规则

文件名格式：`ui_screenshot_{页面}_{设备}_{时间戳}.png`

示例：
- `ui_screenshot_login_web_20250126103000.png` - 登录页面Web端
- `ui_screenshot_home_mobile_20250126103100.png` - 首页移动端

### 推荐的切图清单

#### 登录页面
- [ ] Web端 - 初始状态
- [ ] Web端 - 输入框聚焦状态
- [ ] Web端 - 错误提示状态
- [ ] 移动端 - 初始状态
- [ ] 移动端 - 键盘弹出状态

#### 首页
- [ ] Web端 - Tab1激活状态
- [ ] Web端 - Tab2激活状态
- [ ] 移动端 - Tab1激活状态
- [ ] 移动端 - Tab2激活状态
- [ ] 移动端 - 侧边栏打开状态

### 切图技巧

1. **高清截图**：工具会自动生成2倍高清截图，适合设计稿使用

2. **完整状态**：确保所有交互元素都处于理想状态（如输入框有内容、按钮有悬停效果等）

3. **移动端侧边栏**：点击菜单按钮打开侧边栏后再截图

4. **批量切图**：建议按照清单逐一完成，避免遗漏

## 技术栈

- HTML5
- CSS3（渐变、动画、响应式）
- Vanilla JavaScript
- html2canvas（截图功能）
- Font Awesome（图标）

## 浏览器兼容性

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## 注意事项

1. 切图功能需要网络连接（加载html2canvas库）
2. 截图时会短暂显示加载动画
3. 建议使用现代浏览器以获得最佳效果
