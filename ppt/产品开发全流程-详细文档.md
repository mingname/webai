## 环节二：UI视觉设计规范

### 设计规范要点

#### 🎨 色彩方案
- 主色调：紫色渐变系
  - 主色1：#667eea（浅紫色）
  - 主色2：#764ba2（深紫色）
  - 背景色：#f5f7fa（浅灰）
  - 成功色：#10B981（绿色）
  - 错误色：#EF4444（红色）

#### 📏 尺寸规范
- Web端：卡片宽度400px，最大宽度420px
- 移动端：宽度375px，模拟iPhone尺寸
- 响应式断点：移动端 < 768px，平板端 768-1199px，Web端 ≥ 1200px

#### 🔤 字体规范
- Web端：标题24-32px，正文14-16px
- 移动端：标题20-24px，正文13-14px
- 字体：Font Awesome 6.4 图标

#### 📐 间距与圆角
- 卡片间距：Web端20px，移动端15px
- 圆角：Web端15-20px，移动端12-15px
- 按钮圆角：Web端10-12px，移动端8-10px

**交付文件：** 设计原型.html
- 包含完整的Web端和移动端设计规范
- 文件位置：`d:/uisheji/设计原型.html`

**截图标注：**
- 设计原型文件展示了完整的界面设计规范
- 包含Web端和移动端双端设计稿
- 涵盖登录页、首页（项目Tab、数据统计Tab）等核心页面

## 环节三：Web代码生成

### 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| 🖥️ Vue 3 | Composition API | 响应式框架 |
| ⚡ Vite 5 | 最新版 | 快速构建、开发体验优化 |
| 📦 Pinia | 2.1 | 状态管理、数据持久化 |
| 🛣️ Vue Router | 4.2 | 路由管理、页面导航 |
| 📡 Axios | 1.6 | HTTP客户端、API请求 |

### 项目结构

```
src/
├── views/              # 页面组件
│   ├── LoginView.vue    # 登录页面
│   ├── HomeView.vue     # 首页
│   └── RegisterView.vue # 注册页面
├── api/                # API接口封装
│   └── login.js        # 登录相关API
├── stores/             # Pinia状态管理
│   └── user.js         # 用户状态
├── router/             # 路由配置
│   └── index.js        # 路由定义
├── assets/             # 静态资源
└── App.vue            # 根组件
```

### Web端功能实现

#### 🔐 用户认证模块
- **LoginView.vue** - 登录页面
  - 用户名/密码表单验证
  - 密码显示/隐藏切换
  - 记住密码功能
  - JWT Token认证
  - 错误提示弹窗

- **RegisterView.vue** - 注册页面
  - 用户注册表单
  - 用户名唯一性检查
  - 注册成功自动登录

#### 🏠 首页功能
- **HomeView.vue** - 首页
  - Tab切换（项目/数据统计）
  - 侧边导航栏（移动端）
  - 用户信息展示
  - 响应式布局
  - 退出登录功能

#### 📡 数据交互
- Axios HTTP客户端封装
- API接口统一管理
- 请求拦截器（Token注入）
- 错误统一处理

#### 🎨 交互体验
- Vue Transition过渡动画
- 加载状态提示
- 错误弹窗（Modal组件）
- 表单实时验证

**截图标注：**
- `src/views/LoginView.vue` - 实现了完整的登录表单和交互逻辑
- `src/views/HomeView.vue` - 实现了首页Tab切换、侧边导航等功能
- 代码位置：`d:/uisheji/src/views/`

## 环节四：APP代码生成

### 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| 🤖 Kotlin | 最新版 | 现代Android开发、类型安全 |
| 🏗️ MVVM | - | 架构模式、分层设计 |
| 📡 Retrofit | 2.9.0 | 网络请求、REST API |
| 🗄️ LiveData | AndroidX | 数据观察、生命周期感知 |
| 🖼️ Glide | 4.16.0 | 图片加载、缓存管理 |

### 项目结构

```
app/src/main/java/com/htnova/aitest/
├── ui/                 # UI层
│   ├── LoginActivity.kt       # 登录页面
│   ├── HomeActivity.kt       # 首页
│   └── RegisterActivity.kt    # 注册页面
├── repository/         # 数据仓库层
│   └── AuthRepository.kt      # 认证相关数据操作
├── network/           # 网络层
│   └── NetworkConfig.kt      # Retrofit配置
├── model/             # 数据模型
│   └── ApiModels.kt          # API数据模型
├── adapter/           # 适配器
│   ├── ProjectAdapter.kt       # 项目列表适配器
│   └── StatAdapter.kt         # 统计列表适配器
└── utils/             # 工具类
    └── TokenManager.kt        # Token管理
```

### Android端功能实现

#### 🔐 用户认证
- **LoginActivity** - 登录页面
  - 用户名/密码输入
  - 记住密码功能（SharedPreferences）
  - Token本地存储
  - 用户信息缓存

#### 🏠 首页展示
- **HomeActivity** - 首页
  - Tab切换功能
  - RecyclerView列表展示
  - 项目卡片列表（ProjectAdapter）
  - 数据统计列表（StatAdapter）
  - 侧边导航栏
  - 退出登录

#### 📡 网络层
- Retrofit + OkHttp网络请求
- Gson数据解析
- 协程（Coroutines）异步请求
- 请求拦截器（Token注入）

#### 🖼️ UI组件
- Material Design组件
- Glide图片加载
- 自定义RecyclerView适配器
- 动画效果

**API集成：**
- 后端API地址：`http://115.190.154.26:3001/api/`
- 接口文档：见 `PRD.md` 第6章

**截图标注：**
- `LoginActivity.kt` - Android登录页面实现
- `HomeActivity.kt` - Android首页实现
- 代码位置：`d:/uisheji/asspace/app/src/main/java/com/htnova/aitest/`
