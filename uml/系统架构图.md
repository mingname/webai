# 系统架构图 (System Architecture Diagram)

## 1. 概述

UI设计管理系统采用前后端分离架构，前端使用 Vue 3 + Vite，后端使用 Node.js + Express，数据库使用 MySQL。

**架构模式：**
- 前后端分离
- RESTful API
- JWT 认证
- MVC 设计模式

## 5. 认证流程架构图

```mermaid
sequenceDiagram
    participant Client as 客户端
    participant Frontend as 前端应用
    participant Backend as 后端API
    participant Database as 数据库
    participant JWT as JWT服务
    
    rect rgb(200, 220, 240)
        Note over Client,JWT: 登录流程
        Client->>Frontend: 输入用户名和密码
        Frontend->>Backend: POST /api/login {username, password}
        Backend->>Database: 查询用户
        Database-->>Backend: 返回用户信息
        Backend->>Backend: 验证密码 (bcrypt)
        Backend->>JWT: 生成JWT token
        JWT-->>Backend: 返回token
        Backend-->>Frontend: 返回 {token, userInfo}
        Frontend->>Frontend: 保存token到localStorage
        Frontend-->>Client: 跳转到首页
    end
    
    rect rgb(220, 240, 200)
        Note over Client,JWT: 访问受保护资源
        Client->>Frontend: 访问需要认证的页面
        Frontend->>Frontend: 从localStorage读取token
        Frontend->>Backend: GET /api/user/info<br/>Authorization: Bearer {token}
        Backend->>JWT: 验证token
        JWT-->>Backend: 验证通过
        Backend->>Database: 查询用户信息
        Database-->>Backend: 返回用户信息
        Backend-->>Frontend: 返回用户信息
        Frontend-->>Client: 显示用户信息
    end
    
    rect rgb(240, 200, 220)
        Note over Client,JWT: 退出登录
        Client->>Frontend: 点击退出按钮
        Frontend->>Frontend: 清除localStorage中的token
        Frontend->>Backend: POST /api/logout (可选)
        Backend-->>Frontend: 返回成功
        Frontend-->>Client: 跳转到登录页
    end
```
