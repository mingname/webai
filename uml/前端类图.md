# 前端类图 (Frontend Class Diagram)

## 1. 概述

前端采用 Vue 3 组合式 API (Composition API)，使用 Pinia 进行状态管理，Vue Router 进行路由管理。

**技术栈：**
- Vue 3.4
- Vite 5.0
- Pinia 2.1
- Vue Router 4.2
- Axios 1.6

**路由配置：**

| 路径 | 名称 | 组件 | 需要认证 | 说明 |
|------|------|------|---------|------|
| / | - | 重定向到 /login | 否 | 默认路由 |
| /login | Login | LoginView.vue | 否 | 登录页面 |
| /register | Register | RegisterView.vue | 否 | 注册页面 |
| /home | Home | HomeView.vue | 是 | 首页 |

**路由守卫逻辑：**
```javascript
beforeEach((to, from, next) => {
  // 1. 设置页面标题
  document.title = to.meta.title || 'Vue应用'
  
  // 2. 检查是否需要认证
  if (to.meta.requiresAuth && !userStore.isLoggedIn) {
    next('/login')  // 未登录，跳转到登录页
  } else if ((to.path === '/login' || to.path === '/register') && userStore.isLoggedIn) {
    next('/home')  // 已登录，跳转到首页
  } else {
    next()  // 放行
  }
})
```

**API配置：**

| 配置项 | 值 | 说明 |
|--------|-----|------|
| baseURL | '/api' | API基础路径 |
| timeout | 5000 | 请求超时时间（毫秒） |

**请求拦截器：**
```javascript
config => {
  // 从localStorage获取token
  const token = localStorage.getItem('token')
  if (token) {
    config.headers['Authorization'] = `Bearer ${token}`
  }
  return config
}
```

**响应拦截器：**
```javascript
response => {
  const res = response.data
  console.log('API 成功响应:', res)
  return res
}
```

**错误拦截器：**
```javascript
error => {
  if (error.response && error.response.data) {
    // 后端返回错误，返回错误数据
    return error.response.data
  } else if (error.response) {
    // 有响应但没有数据
    return {
      code: error.response.status,
      message: `请求失败 (${error.response.status})`,
      data: null
    }
  } else {
    // 网络错误
    return Promise.reject(error)
  }
}
```

## 4. 时序图

### 4.1 用户登录流程

```mermaid
sequenceDiagram
    participant U as 用户
    participant L as LoginView
    participant S as UserStore
    participant A as APIService
    participant AX as AxiosInstance
    participant API as 后端API
    participant LS as LocalStorage
    
    U->>L: 输入用户名和密码
    U->>L: 点击登录按钮
    L->>S: login({username, password, rememberPassword})
    S->>A: login({username, password})
    A->>AX: post('/login', {username, password})
    AX->>AX: 请求拦截器添加token（如果存在）
    AX->>API: POST /api/login
    API-->>AX: {code: 200, data: {token, userInfo}}
    AX->>AX: 响应拦截器处理
    AX-->>A: {code: 200, data: {token, userInfo}}
    A-->>S: {code: 200, data: {token, userInfo}}
    S->>S: this.token = token
    S->>S: this.userInfo = userInfo
    S->>LS: setItem('token', token)
    S->>LS: setItem('userInfo', JSON.stringify(userInfo))
    alt 记住密码
        S->>LS: setItem('savedUsername', username)
        S->>LS: setItem('savedPassword', password)
    end
    S-->>L: {success: true, message: "登录成功"}
    L->>L: loading = false
    L->>Router: push('/home')
```

sequenceDiagram
